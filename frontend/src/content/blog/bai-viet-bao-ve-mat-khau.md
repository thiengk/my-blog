## 🔐 Chào mừng bạn đã mở khóa bài viết!

Nếu bạn đang đọc nội dung này, nghĩa là bạn đã nhập đúng mật khẩu. Tính năng bảo vệ bài viết bằng mật khẩu đang hoạt động chính xác.

## 📋 Nội dung kiểm tra

### 1. Định dạng văn bản

Đây là đoạn văn bản thông thường. **Đây là chữ đậm.** *Đây là chữ nghiêng.* `Đây là inline code.`

### 2. Danh sách

- Mục thứ nhất
- Mục thứ hai
- Mục thứ ba

### 3. Code block

```javascript
function greet(name) {

}

greet('Người đọc');
```

Bài viết này xác nhận rằng:
1. Mật khẩu trong frontmatter hoạt động đúng

3. Các định dạng Markdown render chính xác sau khi mở khóa
